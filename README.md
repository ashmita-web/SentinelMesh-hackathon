# SentinelMesh: Autonomous Quantum-Safe Cyber Defense Fabric

## Project Overview

SentinelMesh is an innovative frontend prototype for an autonomous cyber defense system protecting legacy industrial infrastructure. Combining AI-driven anomaly detection, quantum-safe cryptography, federated learning, and decentralized trust, SentinelMesh delivers real-time, verifiable protection designed for critical infrastructure with legacy systems.

Explore the project and live demo here:  
[SentinelMesh Live Demo](https://sentinelmesh.vercel.app/)

---

## Features

### Authentication
- Signup, login, email verification, password reset via Supabase.
- Protected routes ensuring only authenticated users can access sensitive pages.

### Dashboard & Pages
- **Home:** Landing page with platform overview.
- **Demo Console:** Interactive demo environment.
- **Digital Twin:** Simulation and modeling of networks.
- **Anomalies:** Detection and monitoring of anomalies.
- **Federation:** Collaborative network management.
- **Trust & Audit:** Monitoring and auditing tools.
- **Documentation & About:** Guides and project information.

### Notifications
- Toast notifications for user actions like sign-in, sign-up, and password reset.

### Responsive UI
- Modern interface with light/dark theme support.

## Getting Started

You can work on this project locally or via your preferred development tools.

### Local Development

1. **Clone the repository:**
```bash
git clone https://github.com/ashmita-web/sentinelmesh.git
```
3. **Navigate to the project directory:**  
```
cd sentinelmesh
```

4. **Install dependencies:**  
```
npm install
```

5. **Start the development server:**  
```
npm run dev
```

The development server supports hot reloading for a seamless experience.

## Technologies Used

- **Vite** – Lightning-fast frontend build tool  
- **TypeScript** – Strongly typed JavaScript  
- **React** – Component-based UI library  
- **Tailwind CSS** – Utility-first CSS framework  
- **shadcn-ui** – Accessible and customizable React UI components  
- **Lucide-react** – Icon components  

---

## Project Structure Highlights

- `src/components/ui` – Reusable UI components such as Card, Badge, Button, etc.  
- `src/lib` – Utility and service libraries including API client stubs  
- `src/pages` – Application pages, including the About page featuring mission and roadmap  
- `src/hooks` – Custom React hooks for state and logic management
